# Graphix

![Demo](docs/demo.gif)

## Dependencies

```bash
brew bundle
```

## Build

```bash
make
```

## Run

```bash
./graphix <model>
```

## Controls

| Key | Action |
| --- | --- |
| **ESC** | Close window |
| **SPACE** | Toggle wireframe |
| **W/A/S/D** | Pan camera |
| **Scroll** | Zoom in/out |
| **R** | Reset camera to default position |
| **+** | Increase subdivision level |
| **-** | Decrease subdivision level |


## Requirements

- macOS
