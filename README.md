# Graphix

![Demo](docs/demo.gif)

## Dependencies

```bash
brew bundle
```

## Build

```bash
make
```

## Run

```bash
./graphix <model>
```

## Controls

| Key | Action |
| --- | --- |
| **ESC** | Close window |
| **SPACE** | Toggle wireframe |
| **+** | Increase subdivision level |
| **-** | Decrease subdivision level |

Subdivision controls are intended for performance testing to measure optimisation effects

## Requirements

- macOS
